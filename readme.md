# Patient Visit Tracker

A full-stack application for managing and tracking patient visits.

## Overview

This repository contains both backend and frontend applications for the Patient Visit Tracker system.

## Projects

- **Backend**: NestJS REST API for patient visit management
- **Frontend**: React with React Router for the user interface

## Getting Started

### Backend Setup

```bash
cd backend
pnpm install
pnpm run start:dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
